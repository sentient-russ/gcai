# GroupConscience.ai

### Overview:
<p>GroupConscience.ai is an AI chatbot that utilizes AA literature combined with user contributed comments to produce its responses.  The system requires membership to cast votes.  Only comments that have been liked by a minimum threshold are included in the AI responses.</p>
<p>The system utilizes .Net, C#, SignalR, Javasctript, CSS, HTML, ChromaDb, Python, and MySQL as it's high level technologies.  It is configured to run in a Linux environment. </p>
